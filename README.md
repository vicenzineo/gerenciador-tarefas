# Gerenciador de Tarefas (C++)

Um gerenciador de tarefas simples feito em C++ para rodar no terminal.

## Sobre o projeto

Este projeto permite gerenciar uma lista de tarefas com operações basicas:

- Adicionar tarefa
- Excluir tarefa
- Listar tarefas
- Alterar estado de uma tarefa

Cada tarefa possui:

- `titulo`
- `estado` (ex.: `Pendente`, `Em andamento`, `Concluida`)
